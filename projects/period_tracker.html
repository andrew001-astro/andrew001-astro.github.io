Period Tracker — An Android Mobile App for Tracking Period

Year

2022

An Android App to track menstruation by logging the dates when it occured. The user could also log the days when she taken meds and add some description. The logged data will be displayed in a calendar corresponding to their dates.

I used Javascript and some Java to code, React-native for the framework, Android for the OS, Vistual Studio for the IDE, Adobe photoshop for the Logo and Splash Screen, SQLite for offline Database, Mongosh Data API for Online Collection.
